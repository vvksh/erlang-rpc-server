# Erlang RPC server

This is a toy exercise to learn Erlang. I will write a rpc server in Erlang.


## How to run

### Compile
```shell script
erlc src/tr_server.erl
```

### Run
- Open erlang shell
```shell script
erl

```




## Install Erlang
```shell script
brew install erlang
```


## IDE
I am using IntelliJ with erlang plugin.
